# MebelFab × SketchUp — заглушка

> Подсистема: **MebelFab × SketchUp**
> Namespace: `NN::MebelFab::Skp` (планируется)
> Attribute dictionary: `nn_mebelfab` (планируется)

Здесь будут жить:

- Генераторы листовых деталей (ЛДСП/фанера фиксированных толщин: 16/18/22 мм).
- Параметрические компоненты «Шкаф / Полка / Тумба» (DC-параметризация).
- Кромка (оклейка/безкромка) — атрибут на ребре face.
- Фурнитура (петли, евровинты, минификсы) — позиционирование + cut-list.
- Связь с OCL только через имена материалов (без импорта GPLv3-кода).

**Когда сюда добавлять код**: запрос пользователя «в MebelFab в SketchUp …» / «сгенерируй шкаф / тумбу / полку» / «положи деталь из плиты».

См. также [../README.md](../README.md) — общие правила MebelFab-ветки.
