# MebelFab × LayOut — заглушка

> Подсистема: **MebelFab × LayOut**
> Namespace: `NN::MebelFab::Layout` (планируется)

Здесь будут жить:

- Сборочные чертежи мебели (виды сверху / спереди / сбоку, разрезы).
- Спецификации (BOM): материал, толщина, размер, кромка, шт.
- Cut-list-ы листового раскроя (через OCL Cutting Diagrams; OCL даёт layout картинки, мы их вставляем в LayOut + добавляем title block).
- Карты сборки с маркировкой деталей.

**Когда сюда добавлять код**: запрос пользователя «в MebelFab в LayOut …» / «чертёж шкафа в LayOut» / «карта раскроя плиты на лист».

См. также [../README.md](../README.md) — общие правила MebelFab-ветки.
