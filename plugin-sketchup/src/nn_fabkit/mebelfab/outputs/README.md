# MebelFab × Outputs — заглушка

Зонтик для будущих output-форматов мебельной ветки. Когда заказчик даст требования — здесь могут жить:

- **Карты раскроя плит** (PNG/PDF/DXF) — через OpenCutList Cutting Diagrams (только канонические имена материалов, не код, ADR-016).
- **Спецификация фурнитуры** — CSV/XLSX для поставщика.
- **G-code / DXF для ЧПУ** под кромкооблицовочные станки и сверлильно-присадочные центры — если у заказчика появится такое оборудование.

**Текущее состояние**: пусто. Мебельная ветка целиком ждёт вводных от заказчика.

См. также [../README.md](../README.md) — общие правила MebelFab.
