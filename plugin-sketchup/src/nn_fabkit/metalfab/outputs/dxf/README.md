# MetalFab × Outputs × DXF — заглушка

Здесь будет жить генератор DXF для:

- **Раскрой листового материала** (ГОСТ 19903 г/к, ГОСТ 16523 х/к и т.п.) — для плазмы / лазера / гидрорезки. DXF с замкнутыми контурами + bend-lines если есть гибы.
- Возможно — DXF из `Лист.skp` компонентов с пробивкой отверстий.

**Текущее состояние**: пусто. Заказчик ещё не дал требований по DXF — приоритет сейчас IGES для трубы. Когда будет техзадание (от какого CAM, какие entities, версия DXF) — сюда лягут модули.

См. также [../README.md](../README.md) — общая навигация по MetalFab Outputs.
